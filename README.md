# andrewrowe.net
The source code for my [personal website](https://andrewrowe.net).

## Requirements
1. [Pandoc](https://pandoc.org/)
2. [Python3 (for local dev)](https://www.python.org/downloads/)

## Build and Deploy
```bash
# To run the local dev server
bash local_dev.bash

# Build
sh build.sh

# Deploy (with your hostname)
HOSTNAME=host.name sh deploy.sh
```
